# 系统要求

要[运行 FiveM](#)，你的系统必须满足[原始游戏的最低要求](#)。

FiveM 需要 **完全更新** 的 Windows 8.1、Windows 10 或 Windows 11 版本。较旧的操作系统可能无法正常运行。通常建议你升级到 Windows 10，因为它提供了最佳的体验。

更多细节：

| 最低要求                                | 推荐配置                                  |
|------------------------------------------|-------------------------------------------|
| **CPU**                                  |                                           |
| Intel Core 2 Q6600 @ 2.40GHz /           | Intel Core i5 3470 @ 3.2GHz /             |
| AMD Phenom 9850 @ 2.5GHz                 | AMD X8 FX-8350 @ 4GHz                    |
|                                          |                                           |
| **GPU**                                  |                                           |
| NVIDIA 9800 GT 1GB /                     | NVIDIA GTX 660 2GB /                      |
| AMD HD 4870 1GB / Intel HD GT2           | AMD HD7870 2GB                            |
|                                          |                                           |
| **内存**                                 |                                           |
| 8GB（4GB 也可能工作）                    | 16GB                                      |
|                                          |                                           |
| **硬盘**                                 |                                           |
| 105GB + ~4GB                             | 105GB + 8GB                               |

### 注意事项：
1. 可能不适用于某些较旧的 AMD 笔记本 GPU。更多信息请点击[这里](#)。
2. 原始游戏需要 105GB + 额外的 FiveM 缓存空间。
