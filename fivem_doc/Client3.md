# 安装 FiveM

安装 FiveM 非常简单。通常只需要[下载 FiveM](#)，然后运行下载的文件即可。对于需要更多细节的人，请阅读以下内容。

## 安装 #

遇到问题？请确保你的系统符合[最低要求](#)，并阅读[客户端问题手册](#)。

- 如果在启动 FiveM 时遇到任何杀毒软件问题，你可能需要提交[支持工单](#)。
- 确保你已经安装并更新了 GTA V。
- 从官网上[下载 FiveM](#)。
- 运行 FiveM.exe。如果你在空文件夹中运行安装程序，FiveM 将安装在那里。否则，它将安装在 `%localappdata%\FiveM`。
- 从 Windows 开始菜单启动 FiveM。

## 卸载

- 如果你在空文件夹中安装了 FiveM，只需删除该文件夹即可。
- 否则：
  1. 按下键盘上的 `WinKey+R`
  2. 输入 `appwiz.cpl`
  3. 找到 FiveM，右键单击并选择卸载

- 或者：
  1. 按下键盘上的 `WinKey+R`
  2. 在打开的对话框中输入 `%localappdata%`
  3. 按下回车
  4. 删除 FiveM 文件夹

- 这样就全部完成了！
- 如果你想重新安装 FiveM，再次运行 FiveM.exe 即可。
