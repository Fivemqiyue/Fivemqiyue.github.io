# 设置服务器

本页将向你展示如何在 Windows、Linux、txAdmin 上运行 Cfx.re 平台服务器（也称为 “FXServer”），甚至还有 ZAP-Hosting 的指南！它们都可以在下方的[可用指南](#)中找到。

遇到运行服务器的问题？请查看 [服务器问题常见问题解答](#)，使用 Discord 的 `#server-talk` 频道，或者在论坛的服务器讨论子类别中创建一个主题。

## 在你开始之前

如果你还没有这样做，请在 [Cfx.re Keymaster](#) 上注册一个免费的许可密钥。

当被要求输入 IP 时，该 IP 应为你将首先使用该密钥的 **公共 IP**。之后，该密钥可用于任何 IP 的单个服务器实例。
