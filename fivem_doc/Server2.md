# 使用 ZAP-Hosting 设置服务器

## ZAP-Hosting 设置步骤

1. 访问 [ZAP-Hosting 网站](#) 并登录你的账户（如果没有账户，请注册）。
2. 点击屏幕左上角的“[租用服务器](#)”（导航栏）。
3. 将显示一个对话框，你可以选择你的产品。我们将使用 “Cloud Gameserver”，所以点击 “Gameserver”，然后选择 “Cloud Gameserver”。
4. 显示一个新页面，向下滚动直到你找到“[查找游戏](#)”部分，选择 “FiveM”。我们将使用 **FiveM Mod (Windows)**，所以点击它。你也可以点击此[链接](#)。
5. 选择服务器位置，在这个示例中我们将使用位于德国的服务器。
6. 向下滚动并根据你的需要自定义服务器，记住如果要使用超过 48 个插槽，你将需要 [FiveM Elsmert Club Argentum](#) 会员或更高级别的会员，该会员可以通过 Patreon 获得。

   - 如果你打算通过 Patreon 获得订阅，则应使用与 Cfx.re 账户相同的电子邮件地址注册服务器，以便你的计划自动设置和配置。
   - 当使用 ZAP-Hosting 托管服务器时，你需要添加与你的 Cfx.re 账户关联的许可证密钥到你的服务器实例。你可以在[这里](#)找到这些指南。

7. 选择你首选的支付方式并同意条款，记得保持登录状态。
8. 确认订单后，服务器将启动。一旦服务器启动，将显示一个 “初始安装” 对话框。点击 “安装”。
9. 你现在应该可以访问你的游戏面板，显示的 IP 地址将用于访问你的服务器。
10. 复制 IP 并启动 FiveM。一旦 FiveM 启动，按下键盘上的 F8 打开游戏控制台，输入 `connect [你的服务器IP]`，你应该能够连接到正在运行的服务器。
